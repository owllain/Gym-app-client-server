Manual de uso:
1. Conexión Cliente ➞ Servidor. El primer paso para probar la solución es habilitar
la conexión del servidor al cliente. Esta conexión no es necesaria para poder
conectarse a la base de datos.
2. Proceder con los registros básicos. Creación de un cliente y una sede.
3. Registrar Sedes de Clientes y, por último, a las Sedes activas, asignarles los
Cupos disponibles para las reservaciones.
4. Una vez teniendo clientes registrados en la base de datos y estos mismos
teniendo como mínimo una afiliación a una sede, con el número de identificación
se puede ingresar por medio de la aplicación cliente.
5. Una vez conectado a la aplicación cliente, para poder realizar las reservaciones
el cliente debe poseer una afiliación a una Sede y esa misma Sede debe tener
cupos disponibles.
6. Para salir, una vez el cliente realicé la desconexión, debe desconectar el
servidor.
